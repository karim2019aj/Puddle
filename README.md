# Marketplace
Simple online marketplace where people can buy and sell items.






## Tech Stack

**Client:**  TailwindCSS

**Server:** Django


## Features

- Item detail
- Signing up
- Logging in
- Adding items
- Dashboard
- Delete items
- Edit items
